# 🔍 Michi-Explorer: Buscador Global Premium

Michi-Explorer es nuestra herramienta de búsqueda multimodal, diseñada para navegación instantánea en todo el ecosistema de **Michicondrias**.

## Modos de Operación

### 1. Activador de Topbar 🛰️
Una barra de búsqueda integrada en la cabecera con **Glassmorphism** que sirve como disparador visual.

### 2. Paleta de Comandos (Ctrl+K) ⌨️
Inspirado por las mejores interfaces para desarrolladores, permite:
- Abrir el buscador desde cualquier sección sin usar el ratón.
- Un panel flotante con desenfoque de fondo profundo (`blur(20px)`).

## Capacidades de Búsqueda 🛰️
Michi-Explorer indexa y previsualiza en tiempo real:
- **Mascotas**: Nombre, raza y especie.
- **Clínicas**: Nombre y ubicación.
- **Productos**: Nombre y precio estimado.

## Control por Teclado 🕹️
- `Ctrl + K`: Abrir/Cerrar buscador.
- `↑ / ↓`: Navegar entre resultados de diferentes secciones.
- `Enter`: Navegar directamente a la ficha del resultado seleccionado.
- `Esc`: Salida rápida del modo exploración.
